• ∅ ∈ τ and X ∈ τ
• ⋃_{i∈I} U_i ∈ τ for any family {U_i}_{i∈I} ⊆ τ
• ⋂_{i=1}^n U_i ∈ τ for any finite subfamily {U_i}_{i=1}^n ⊆ τ
