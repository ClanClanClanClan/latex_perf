## CI gate

`.github/workflows/proof.yml` builds the proof tree with `dune build`,
then greps for `Admitted.` / `admit.` / `Axiom` / `Parameter`. Exit 1
on any match. Runs on every push and PR; cannot be bypassed.

Run locally:

```bash
OPAMSWITCH=l0-testing opam exec -- dune build @coq
```
