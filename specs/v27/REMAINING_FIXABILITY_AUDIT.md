# Remaining Rule Fixability Audit (v27.1.20)

Classification of the 391 diagnose-only rules (those with NO `fix:` token in the catalog) into auto-fixable / candidate-able / genuinely-diagnose-only.

**Totals:** 10 auto-fix · 170 candidate · 208 diagnose-only (of 391).

## Assessment

Of the 391 remaining diagnose-only rules, only 10 are genuinely clean, corruption-free auto-fixes and 170 are review-gated candidates; the other 211 (54%) are truly un-fixable (layout/log-derived LAY/COL/META, external-data BIB/REF/DOC, meta-diagnostics PRT/EXP/STRUCT-005, and rename/authoring tasks). So on high-certainty MECHANICAL work the user is essentially wrong: the auto-fix universe is ~159 shipped + 10 remaining ≈ 169, meaning ~94% is already done and only a single small batch of clean producers is left (MATH-072/091, SCRIPT-018/021, CHEM-004, CJK-009, STRUCT-003, ENC-006, DE-006, BIB-010). On CANDIDATES the user is right: the universe is ~18 shipped + 170 remaining = 188, so only ~10% is touched — the review-gated space is barely explored. But that pool is lower-value and lower-certainty: it needs a review/approval UX, many entries are document-wide stylistic normalizations (STYLE/MOD/LANG/PT/quotes) or cross-reference renames (REF/BIB keys) that carry real corruption risk, and per-rule payoff diminishes sharply versus the mechanical producers already shipped. Net: "barely touched a fourth" holds only for the candidate track; the certain auto-fix track is nearly exhausted.

## Auto-fix (10) — finishes the mechanical track

- **BIB-010** — month = {N}/"N"/N with N in 1..12 -> replace with the fixed macro jan..dec via a deterministic 12-entry table.
- **CHEM-004** — Wrap single-char charge superscript in braces: ^- → ^{-}, ^+ → ^{+}; always render-equivalent and safe in math.
- **CJK-009** — Delete the Western space between two adjacent CJK glyphs (CJK has no inter-word space); same shape as shipped CJK-008/015.
- **DE-006** — Swiss-German locale: unconditional replace ß (U+00DF) → ss; deterministic and locale-correct in all positions.
- **ENC-006** — Overlong UTF-8 sequence decodes to an unambiguous codepoint; re-encode in canonical shortest form — deterministic.
- **MATH-072** — Predefined operator wrapped in \operatorname → dedicated command: `\operatorname{sin}`→`\sin`, `\operatorname{log}`→`\log`, etc. Fixed lookup-table swap, corruption-free.
- **MATH-091** — `\operatorname{det}` → `\det` (and siblings that have a predefined command). Fixed deterministic swap, corruption-free — the canonical shipped-producer shape.
- **SCRIPT-018** — Add braces: ^\circ → ^{\circ} within math segments — output-identical, always safe.
- **SCRIPT-021** — Reorder _{b}^{c} → ^{c}_{b} within math — TeX renders identically, always safe, deterministic.
- **STRUCT-003** — Replace tab characters with spaces (verbatim-aware, as shipped VERB-002); deterministic whitespace edit.
